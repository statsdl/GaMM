GaMM
====

``gamm`` provides GaMMixer utilities for GARCH-assisted high-frequency
volatility forecasting.

Installation
------------

.. code-block:: bash

   pip install gamm

Optional model and tuning dependencies:

.. code-block:: bash

   pip install "gamm[full]"

Synthetic benchmark:

.. code-block:: bash

   python examples/run_synthetic_benchmark.py
